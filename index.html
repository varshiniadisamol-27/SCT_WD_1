About This Project

This responsive landing page was developed using HTML, CSS, and JavaScript as part of my Web Development Internship at SkillCraft Technology.

The project features an interactive navigation menu with hover effects, a fixed navigation bar, dynamic color changes on scroll, and a responsive layout designed to provide a smooth user experience across different devices.

Through this project, I gained hands-on experience with DOM manipulation, event handling, CSS transitions, and responsive web design.
